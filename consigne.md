Application de Fruits
Objectif
Créer une application iOS avec SwiftUI qui présente une liste de fruits avec navigation entre plusieurs écrans.

Fonctionnalités Requises
1. Écran Principal (ContentView)
- Un slider
- Un bouton "Start" qui navigue vers la liste des fruits

2. Écran Liste des Fruits (FruitListView)
- Une liste de fruits avec pour chaque élément :
- Une image
    - Un titre (nom du fruit)
    - Une mini-description (1-2 lignes)
- Un bouton "Settings" en haut à droite affiche la page Setting
- Navigation vers l'écran de détail au clic sur un fruit

3. Écran Détail d'un Fruit (FruitDetailView)
- Affichage détaillé du fruit sélectionné :
    - Image en grand
    - Nom du fruit
    - Description complète


Contraintes Techniques
- Utiliser SwiftUI (pas d'UIKit)
- Respecter l'architecture MVVM (Modèle-Vue-VueModèle)
- Le code doit être propre et bien commenté

Contraintes de Conception de projet
- 1 commentaire en français humanizé en début de chaque fichier et d'autres commentaire où il serait utile pas d'autres commentaires

Barème

- Liste des Fruits : 3 points
- Setting : 3 points
